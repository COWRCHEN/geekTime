#### 使用说明

1. git clone https://github.com/chong2230/geekTime.git
2. cd geekTime
3. npm install
4. 由于RN版本更新，需将tools目录下ViewPager.js和DefaultViewPageIndicator.js覆盖node_modules/react-native-viewpager下对应文件
5. react-native link
6. react-native run-ios
